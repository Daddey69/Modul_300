# Git / Versionierung
Git ist eine online Platform für die Versionierung von Programm Code. Hierbei ist der Vorteil, dass Personen gleichzeitig am selben Code als colaboration arbeiten können.

## Befehle
1. git add .
2. git commit -m 'Neuste Version'
3. git push
4. git log
5. git pull
6. git clone
